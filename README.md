# Healthcare Billing Data Analysis (SQL Project)

## Overview
This project analyzes healthcare billing data to understand claim outcomes, revenue patterns, and billing errors. The goal is to simulate real-world healthcare data analysis similar to payer or provider systems.

## Tools Used
- SQL
- Python (Pandas)
- CSV dataset

## Objectives
- Analyze total revenue from billing data
- Identify denied and approved insurance claims
- Detect billing errors and inconsistencies
- Compare insured vs uninsured revenue patterns

## Key Insights
- Certain procedures have higher denial rates
- Billing errors correlate with denied claims
- Insurance status significantly affects total revenue

## Skills Demonstrated
- SQL querying (GROUP BY, aggregations, filtering)
- Data analysis using Python
- Healthcare data interpretation
- Problem-solving and data cleaning thinking
